## Project Task Split

To complete this project as efficiently as possible, we have divided the tasks from our Scrum board between two team members. This split organizes the work logically so that each person can focus on a specific domain of the application.

### Team Member 1: Core Engine & Gameplay Mechanics
*Focuses on log creation, rules, writing UI, and core log interaction.*

**Sprint Todo:**
- #1 Create logs with public or invite-only visibility
- #2 Configure participant limit and turn order for a log
- #3 Add round limit and creative constraints to logs
- #4 Start logs from a seed prompt or blank slate
- #5 Implement writing UI and immutable turn submission
- #6 Color-code each participant's turns in a log

### Team Member 2: Discovery, Social & Additional Features
*Focuses on the feed, social interactions, profile management, and notifications.*

**Sprint Todo:**

**Backlog:**

- #10 Add emoji reactions and comments to completed logs
